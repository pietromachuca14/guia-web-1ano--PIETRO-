O que é HTML, CSS e JS?

HTML

HTML (HyperText Markup Language) estrutura o conteúdo da página com elementos semânticos.

<p>Olá, mundo!</p>

Use para estruturar conteúdo e marcar semântica.

CSS

CSS (Cascading Style Sheets) define o estilo e layout visual do conteúdo HTML.

p { color: var(--primaria); }

Use para estilizar, temas e responsividade.

JavaScript

JavaScript adiciona interatividade e lógica no cliente (navegador).

document.querySelector('p').textContent = 'Olá';

Use para comportamento dinâmico, validações e manipulação DOM.
