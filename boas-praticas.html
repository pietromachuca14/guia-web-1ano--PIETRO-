Boas Práticas Profissionais

  • Use landmarks (header, main, footer) para estruturar.
  • Imagens com alt descritivo.
  • Labels conectados a inputs (for/id).
<label for="nome">Nome</label> <input id="nome"/>

  • Mobile-first e Media Queries
  • Use Flex e Grid para layouts
  • Unidades relativas (rem, %) para tipografia
@media (min-width:600px){ .grid{grid-template-columns:repeat(3,1fr)} }

  • Use convenções como BEM
  • Variáveis CSS para cores e tipografia
  • Comente blocos lógicos
/* .card__title */ .card__title{font-weight:600}

  • Otimizar imagens e usar lazy-loading
  • Evitar JS desnecessário no carregamento
  • Minificar assets em produção
<img src="foto.jpg" loading="lazy" alt="Foto"/>

  • Mensagens de commit claras (feat:, fix:)
  • Funções puras e tratamento de erros
  • Testes unitários quando possível
try{ fetch('/api'); }catch(e){ console.error(e); }
